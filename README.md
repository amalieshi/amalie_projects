## 🎯 Goals & Learning Objectives

### C# Development
- **Web Development**: ASP.NET Core, Blazor, Web APIs
- **Desktop Applications**: WPF, WinUI 3, .NET MAUI
- **Architecture Patterns**: Clean Architecture, CQRS, Microservices
- **Testing**: Unit Testing, Integration Testing, TDD

### Data Science & Machine Learning
- **Python Frameworks**: Django, FastAPI, Flask
- **ML Libraries**: PyTorch, TensorFlow, scikit-learn
- **Data Analysis**: Pandas, NumPy, Matplotlib, Seaborn
- **Deep Learning**: Neural Networks, Computer Vision, NLP

### Frontend Development
- **React Ecosystem**: Hooks, Context, Redux, Next.js
- **Modern JavaScript**: ES6+, TypeScript, Async Programming
- **CSS**: Flexbox, Grid, Responsive Design, Animations

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd amalie_projects
   ```

2. **Choose your focus area**
   - Browse to the relevant folder based on your current learning goal
   - Each major folder contains its own README with specific setup instructions

3. **Set up development environment**
   - Install required SDKs and tools (see individual folder READMEs)
   - Configure your IDE/editor for the technology stack

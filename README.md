## Technology Stack

This repository demonstrates modern development practices across multiple domains:

### Backend Development
- **C# .NET 8+**: ASP.NET Core, Web APIs, desktop applications
- **Python 3.11+**: Django, FastAPI, Flask web frameworks
- **Architecture**: Clean Architecture, CQRS, microservices patterns
- **Testing**: Unit testing, integration testing, TDD practices

### Data Science & Machine Learning
- **ML Frameworks**: PyTorch, TensorFlow, scikit-learn
- **Data Analysis**: Pandas, NumPy, Matplotlib, Seaborn
- **Deep Learning**: Neural networks, computer vision, NLP
- **Notebooks**: Jupyter Lab for data exploration and visualization

### Frontend Development
- **Modern JavaScript**: ES6+, TypeScript, React 18+
- **Responsive Design**: CSS3, Flexbox, Grid layouts
- **State Management**: Context API, Redux patterns
- **Build Tools**: Modern toolchains and deployment pipelines

## Getting Started

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd amalie_projects
   ```

2. **Navigate to your area of interest**
   - Each major folder contains specific setup instructions
   - Review individual README files for project-specific requirements

3. **Development environment setup**
   - Install required SDKs and runtime environments
   - Configure IDE/editor for the chosen technology stack

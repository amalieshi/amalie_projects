# Learning Path Documentation 📚

This folder contains structured learning guides and resources for skill development.

## 📋 Available Learning Paths

### 🔷 C# Development Path
- **Beginner**: Console applications, basic OOP concepts
- **Intermediate**: Web APIs, desktop applications, database integration  
- **Advanced**: Microservices, clean architecture, advanced patterns

### 🐍 Python Development Path
- **Beginner**: Basic syntax, data structures, file operations
- **Intermediate**: Web frameworks, APIs, data manipulation
- **Advanced**: Machine learning, data science, performance optimization

### 📊 Data Science Path
- **Foundation**: Statistics, data analysis, visualization
- **Machine Learning**: Supervised/unsupervised learning, model evaluation
- **Deep Learning**: Neural networks, computer vision, NLP

### ⚛️ Frontend Development Path  
- **Foundation**: HTML, CSS, JavaScript fundamentals
- **Frameworks**: React, Vue.js, modern tooling
- **Advanced**: Performance optimization, testing, accessibility

## 📅 Suggested Learning Schedule

### Month 1-2: Foundations
- Complete C# fundamentals and first console applications
- Learn Python basics and start web development with Django/FastAPI
- Build first frontend project with HTML/CSS/JavaScript

### Month 3-4: Integration  
- Create full-stack applications combining C# APIs with frontend
- Start machine learning fundamentals with scikit-learn
- Build desktop applications with WPF or MAUI

### Month 5-6: Specialization
- Choose focus area: ML/AI, Advanced Web Development, or Cross-platform Development
- Complete advanced projects in chosen specialization
- Contribute to open source projects

## 📖 Recommended Reading Order

1. **C# in Depth** (Jon Skeet) - Advanced C# concepts
2. **Clean Code** (Robert Martin) - Code quality principles
3. **Python Crash Course** (Eric Matthes) - Python fundamentals
4. **Hands-On Machine Learning** (Aurélien Géron) - ML with Python
5. **You Don't Know JS** (Kyle Simpson) - Advanced JavaScript

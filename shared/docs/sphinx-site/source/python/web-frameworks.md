# Python Web Frameworks

## Overview

This section showcases modern Python web development using Django and FastAPI, demonstrating both individual services and orchestrated multi-service architectures.

## Projects Structure

### Django Applications
- **Todo Display**: Full-stack Django application for todo management
- **Todo Orchestrator**: Django-based testing and orchestration platform

### FastAPI Services  
- **Todo API**: High-performance REST API with automatic documentation

## Architecture Highlights

The web frameworks section demonstrates:

- **Microservices Architecture**: Separate Django and FastAPI services
- **Service Orchestration**: Django orchestrator managing FastAPI services
- **Auto-start Functionality**: Automatic server management and process control
- **Production Deployment**: Render-ready configurations with proper CSRF handling
- **Cross-Origin Communication**: Proper CORS setup for service interaction

## Key Technologies

- **Django 6.0**: Modern Python web framework
- **FastAPI**: High-performance API framework with automatic documentation
- **Bootstrap 5**: Responsive UI components
- **SQLite/SQLAlchemy**: Database layer
- **Uvicorn/Gunicorn**: ASGI/WSGI servers for production
- **WhiteNoise**: Static file serving for cloud deployment

## Development Features

- **Live Reload**: Development servers with auto-restart
- **API Testing**: Built-in testing interfaces and debugging tools
- **Health Monitoring**: Service health checks and status monitoring
- **Process Management**: Automatic server startup and PID tracking
- **Request Logging**: Comprehensive API request/response logging
